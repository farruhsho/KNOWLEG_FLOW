## Инструменты Тестирования

### Автоматизированное Тестирование
```bash
# Unit tests
flutter test

# Integration tests
flutter drive --target=test_driver/app.dart

# Widget tests
flutter test test/widget_test.dart
```

### Ручное Тестирование
- Используйте реальные устройства
- Тестируйте на разных версиях ОС
- Проверяйте на разных размерах экранов

### Инструменты Анализа
```bash
# Анализ кода
flutter analyze

# Проверка производительности
flutter run --profile

# Анализ размера
flutter build apk --analyze-size
```
